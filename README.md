# Card Game

This project is a simple game of cards letting the user play a variety of games with a set amount of cards. 

## Version 0.1.0 - Deck Generation 

First version of the program that makes a deck of cards for the user

### Features
- The program now generates a deck
- Deck size can be chosen by the user
